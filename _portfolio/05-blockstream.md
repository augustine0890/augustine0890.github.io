**Company:** SupperTree, South Korea · **Role:** Software Engineer

A scalable system for real-time monitoring and processing of blockchain transaction data from Ethereum and Polygon networks, using a publish-subscribe model for efficient data delivery.

### Key Contributions

- Developed and optimized the architecture to collect blockchain transaction data in real-time from **Ethereum and Polygon**, ensuring comprehensive data coverage and accuracy.
- Engineered a seamless data pipeline using **Kafka** to publish transaction data, stored in **PostgreSQL** and indexed in **OpenSearch / Algolia** for enhanced search functionality.
- Managed the end-to-end data workflow from on-chain collection to storage and indexing, including metadata like images and links for enriched search capabilities.
- Applied transaction pattern analysis and anomaly flagging — methodology directly transferable to payment fraud detection.

### Tech Stack

`Python` · `Golang` · `Apache Kafka` · `Algolia` · `OpenSearch` · `PostgreSQL` · `Oracle` · `Kinesis`
