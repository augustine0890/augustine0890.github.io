**Company:** FPT Software Korea · **Role:** Data Software Engineer

Developed a data analytics and visualization dashboard to monitor and analyze QA/QC data for investment securities, enabling real-time insights for product quality improvement through statistical processing and visual reporting.

### Key Contributions

- Designed and implemented data pipelines to aggregate, preprocess, and analyze large QA/QC datasets using **SQL** and **Java (Apache Commons Math)**, delivering statistical insights for quality enhancement.
- Applied advanced statistical methods to identify trends and anomalies, creating interactive visualizations (pie, bar, line charts) with **Chart.js** for stakeholder decision-making.
- Developed optimized **REST APIs** using **Spring Boot** to deliver real-time statistical reports and dashboards.
- Migrated Oracle → **EDB (PostgreSQL)** for Investment Securities Project Meta, rewriting complex financial SQL queries into ANSI-standard for EDB compatibility while improving query efficiency.

### Tech Stack

`Java` · `Spring Boot` · `SQL` · `Oracle` · `PostgreSQL (EDB)` · `Apache Commons Math` · `Chart.js` · `REST APIs`
